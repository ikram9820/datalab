Setup project environment with python -m venv myenv.

```bash
$ git clone https://github.com/ikram9820/datalab.git
$ cd datalab
$ python -m venv .venv
$ .venv/Scripts/activate
$ pip install -r requirements.txt

$ streamlit run app.py
```
